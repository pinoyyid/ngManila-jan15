A couple of real world changes

  • Need to persist the Todo list
    • Local storage (offline first)
    • A REST cloud service
  • Need add an 'Archive' feature for completed items
    • User should be able to see her completed (unarchived) items
    • User should be able to archive completed items with a checkbox
    • For performance reasons, archived items (of which there will be tens of thousands) should be stored separately from un-archived items (of which there will be < 50)
