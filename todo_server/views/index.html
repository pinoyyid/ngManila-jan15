ngTodo - test server

This is the companion server to the ngManila ToDo app.
This server provides two REST methods to save a new Todo item and to return a list of Todo items

NB. The server has no persistence. This means that although it will return a success 200, the new Task item has NOT been saved, and a subsequent list will always return the same test data


Insert a new Task

Make a POST to /tasks/v1/lists/tasklist/tasks . The POST body should be a JSON object containing a "title" property. Any other properties are ignored.
On success, the object will be echoed back with a populated "id" property. See also https://developers.google.com/google-apps/tasks/v1/reference/tasks/insert


List Tasks

Make a GET to /tasks/v1/lists/tasklist/tasks . On success, an object will be echoed back which contains an "items" property which is a list of Tasks
See also https://developers.google.com/google-apps/tasks/v1/reference/tasks/list
